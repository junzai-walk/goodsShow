# 快速使用指南

## 🚀 快速开始

### 1. 启动项目

项目已经完成初始化和依赖安装，直接运行：

```bash
npm run dev
```

服务器将在 `http://localhost:5173` 启动

### 2. 访问页面

#### 首页
访问 `http://localhost:5173/` 可以看到：
- 三个店铺风格选择卡片
- 后台管理系统入口

#### 前台店铺

**现代简约风格** - `http://localhost:5173/shop1`
- 紫色渐变主题
- 适合时尚、年轻化品牌
- 特色：全屏轮播、网格商品展示

**科技感风格** - `http://localhost:5173/shop2`
- 深色主题 + 霓虹灯效果
- 适合科技、电子产品
- 特色：Glitch 文字效果、发光动画

**传统电商风格** - `http://localhost:5173/shop3`
- 经典红色主题
- 适合综合性电商平台
- 特色：左侧分类、热销排行

#### 后台管理

**数据看板** - `http://localhost:5173/admin` 或 `http://localhost:5173/admin/dashboard`
- 4个实时统计卡片
- 销售趋势折线图
- 分类占比饼图
- 小时销售柱状图
- 热销商品排行榜

**商品管理** - `http://localhost:5173/admin/products`
- 商品列表展示
- 搜索、筛选、排序功能
- 编辑、删除操作

**订单管理** - `http://localhost:5173/admin/orders`
- 订单列表展示
- 状态筛选
- 订单操作

## 🎨 页面功能说明

### 前台功能

1. **商品浏览**
   - 点击分类可筛选商品
   - 商品卡片显示图片、名称、价格、评分、销量
   - 悬停商品卡片有动画效果

2. **轮播图**
   - 自动轮播（5秒切换）
   - Shop2 有指示器可手动切换

3. **导航**
   - 每个店铺页面都有返回首页按钮
   - Footer 有进入后台管理的链接

### 后台功能

1. **侧边栏导航**
   - 点击菜单项切换页面
   - 当前页面高亮显示

2. **数据看板**
   - 实时数据展示（Mock 数据）
   - 图表可交互（悬停显示详情）
   - 响应式布局

3. **商品管理**
   - 搜索框：输入商品名称搜索
   - 分类筛选：点击分类列筛选
   - 排序：点击列标题排序
   - 操作：编辑/删除按钮

4. **订单管理**
   - 状态筛选：点击状态列筛选
   - 订单号复制：点击复制图标
   - 操作：查看/处理/取消

## 📱 启用 autofit.js 自适应

在需要自适应的页面组件中添加：

```typescript
import { useEffect } from 'react';
import { initAutofit, destroyAutofit } from '../../utils/autofit';

const YourComponent = () => {
  useEffect(() => {
    // 初始化自适应，设计稿尺寸 1920x1080
    initAutofit(1920, 1080);
    
    // 组件卸载时清理
    return () => destroyAutofit();
  }, []);
  
  // ...
};
```

## 🔧 常见问题

### 1. 图片加载慢或失败
- 使用的是 Unsplash 图片服务，需要网络连接
- 可以替换为本地图片或其他图片服务

### 2. 图表不显示
- 检查浏览器控制台是否有错误
- 确保 ECharts 正确加载

### 3. 路由跳转不工作
- 检查 React Router 是否正确配置
- 查看浏览器控制台错误信息
