# 电商平台店铺系统 - 项目说明

## 项目概述

这是一个基于 React + TypeScript + Ant Design ProComponents 搭建的现代化电商平台店铺系统，包含三种不同风格的前台店铺展示页面和一个功能完善的后台管理系统。

## 已完成功能

### ✅ 1. 项目初始化
- 使用 Vite + React 19 + TypeScript 初始化项目
- 安装并配置 Ant Design、ProComponents、React Router、ECharts、autofit.js 等依赖

### ✅ 2. Mock 数据创建
- **商品数据** (`src/mock/products.ts`): 包含10个商品，涵盖6个分类
- **Dashboard 数据** (`src/mock/dashboard.ts`): 包含销售趋势、分类占比、地域分布、热销排行等数据

### ✅ 3. 前台店铺页面 - 三种风格

#### 风格1: 现代简约 (`/shop1`)
- **特点**: 清新简洁，渐变色设计
- **组件**: 
  - 渐变色导航栏（紫色渐变）
  - 全屏轮播图展示
  - 分类导航菜单
  - 商品网格布局
  - 商品卡片（图片、名称、价格、评分、销量）
- **交互**: 分类筛选、商品悬停效果、加入购物车

#### 风格2: 科技感 (`/shop2`)
- **特点**: 深色主题，霓虹灯效果，科技蓝配色
- **组件**:
  - 深色导航栏（科技蓝边框）
  - 科技感轮播图（Glitch 文字效果）
  - 炫酷分类导航（悬停发光效果）
  - 商品卡片（图片悬停放大、快速查看）
- **动画**: 脉冲动画、发光效果、渐变按钮

#### 风格3: 传统电商 (`/shop3`)
- **特点**: 经典电商布局，信息丰富
- **组件**:
  - 顶部通知栏（限时活动）
  - 搜索栏 + 热门关键词
  - 左侧分类导航
  - 热销排行榜（Top 5）
  - 商品列表展示
  - 详细的 Footer（购物指南、配送方式等）
- **布局**: 左侧边栏 + 右侧内容区

### ✅ 4. 后台管理系统 (`/admin`)

#### ProLayout 框架
- 使用 `@ant-design/pro-layout` 搭建
- 侧边栏导航菜单
- 顶部导航栏
- 用户头像下拉菜单
- 返回首页快捷按钮

#### 数据可视化看板 (`/admin/dashboard`)
- **实时统计卡片** (4个):
  - 今日销售额（蓝色渐变卡片）
  - 今日订单（绿色渐变卡片）
  - 今日访客（橙色渐变卡片）
  - 总营收（紫色渐变卡片）
  - 每个卡片显示增长率

- **销售趋势图** (折线图):
  - 30天销售额和订单量双轴趋势
  - 面积填充效果
  - 深色主题配色

- **分类销售占比** (饼图):
  - 6个商品分类的销售占比
  - 环形饼图设计
  - 百分比标签显示

- **今日小时销售** (柱状图):
  - 24小时销售分布
  - 渐变色柱状图

- **热销商品排行** (表格):
  - Top 10 商品列表
  - 排名徽章（前三名特殊颜色）
  - 销量、销售额、趋势显示

#### 商品管理 (`/admin/products`)
- 使用 `ProTable` 组件
- 功能:
  - 商品列表展示（图片、名称、分类、价格、库存、销量、评分）
  - 搜索功能（商品名称）
  - 分类筛选
  - 价格、库存、销量排序
  - 编辑、删除操作
  - 新建商品按钮

#### 订单管理 (`/admin/orders`)
- 使用 `ProTable` 组件
- 功能:
  - 订单列表展示
  - 订单状态筛选（待处理、处理中、已完成、已取消）
  - 订单号复制
  - 查看、处理、取消操作

### ✅ 5. 路由配置
- 首页 (`/`): 选择店铺风格
- 三个店铺页面 (`/shop1`, `/shop2`, `/shop3`)
- 后台管理 (`/admin/*`)
- 所有页面都有返回首页/后台的入口

### ✅ 6. 响应式设计
- 使用 Ant Design 的栅格系统
- 适配不同屏幕尺寸（xs, sm, md, lg）
- 移动端友好

## 技术亮点

1. **TypeScript 类型安全**: 所有组件和数据都有完整的类型定义
2. **ProComponents 集成**: 使用 ProLayout、ProTable、ProCard 提升开发效率
3. **ECharts 数据可视化**: 丰富的图表类型，深色主题配色
4. **CSS 动画效果**: 悬停、过渡、渐变等动画效果
5. **Mock 数据完整**: 真实的数据结构，便于后续对接 API
6. **代码结构清晰**: 按功能模块组织，易于维护和扩展

## 项目运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址
http://localhost:5173
```

## 使用 autofit.js

项目已安装 autofit.js，如需在特定页面启用屏幕自适应：

```typescript
import { useEffect } from 'react';
import { initAutofit, destroyAutofit } from '../utils/autofit';

useEffect(() => {
  initAutofit(1920, 1080); // 设计稿尺寸
  return () => destroyAutofit();
}, []);
```
